# Personal Portfolio Website

A fully accessible and semantic personal portfolio website built using HTML5 and CSS. The website follows modern web development standards, WCAG accessibility guidelines, and SEO best practices.

## Features

* Semantic HTML5 structure
* Responsive design
* Accessibility-focused development
* SEO-friendly meta tags
* Keyboard navigable interface
* Accessible contact form
* Technical Skills section
* Coding Profiles section
* Leadership & Responsibilities section
* Projects showcase

## Technologies Used

* HTML5
* CSS3
* Git & GitHub
* Visual Studio Code

## Website Pages

* Home
* index
* About
* Projects
* Contact
* leadership
* codingprofiles
  

## Accessibility Features

* Semantic landmarks (`header`, `nav`, `main`, `section`, `article`, `footer`)
* Skip-to-content link
* Proper heading hierarchy
* ARIA labels
* Keyboard accessibility
* Visible focus indicators
* Mobile-friendly design
